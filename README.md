# crypto-playground

Repositório educacional com exemplos práticos de criptografia aplicada à Segurança da Informação.

## Objetivo
Demonstrar o uso correto de algoritmos criptográficos modernos e clássicos, com foco em boas práticas, aprendizado e análise defensiva.

## Conteúdo
- Criptografia simétrica (AES, modos de operação)
- Criptografia autenticada (AES-GCM)
- Hash e integridade (SHA-256 / SHA-512)
- HMAC e autenticação
- Derivação de chaves (PBKDF2 / Argon2)
- Criptografia assimétrica (RSA / ECC)
- Algoritmos clássicos (uso educacional)
